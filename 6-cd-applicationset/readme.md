## What is an ApplicationSet?
ApplicationSet is a Kubernetes CRD in Argo CD that automates the creation and management of multiple Argo CD Applications using templates and generators. It is commonly used for multi-cluster, multi-environment, and monorepo deployments, enabling scalable and efficient GitOps practices. [References](https://argo-cd.readthedocs.io/en/stable/operator-manual/applicationset/)



## ApplicationSet Generator
In Argo CD, an ApplicationSet Generator is the component that automatically creates multiple Argo CD Applications from a single template. [References](https://argo-cd.readthedocs.io/en/stable/operator-manual/applicationset/Generators/)
